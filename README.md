# Simple To-Do List App

A clean and simple to-do list web application built with vanilla HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Persistent storage using localStorage
- Responsive design
- Task statistics (total, active, completed)
